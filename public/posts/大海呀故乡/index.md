# 秦岭山中的夏日清凉

*2026-07-03*

> 夏日的秦岭-随笔


我终于来到了秦岭。

不是为了拍照、不是为了攀登最高峰，只是想找一个地方，把夏天在城市里被烤得快要融化的心，慢慢融化成一汪清凉。

从西安坐车进去，山路像一条银色的丝带，盘旋在层层叠叠的绿里。车窗外，树叶绿得发亮，像刚被雨水洗过，还在滴水。天空是那种特别蓝的蓝，风一吹，就带着泥土和草木的香气。

我把车停在半山腰的停车场，推开车门，那一刻，风就扑过来，凉得像一杯刚出炉的冰镇绿茶。

“这里比城里凉多了。”我对旁边跟着的小朋友说。他只是点点头，眼睛却亮亮的。

我背上包，沿着石阶往上走。脚下是柔软的落叶，踩上去像踩在云上。路边，野花开得正盛：一种叫“山百合”的花，雪白的花瓣在风里轻轻颤动，像一朵朵从山里飘下来的云；还有红色的“桃儿七”，像一颗颗隐藏在叶底的宝石，静静地低头看着路过的人。

我停下来，蹲下身，轻轻摘了一朵。叶子摸起来凉凉的，带着一点苦涩的清香。我把花插在发间，风一吹，它就飘啊飘，像在告诉我：夏天要来了。

继续往前走，路越来越窄，越来越曲折。突然，前方传来哗啦哗啦的水声。我加快脚步，一眼就看见一条小溪。从山顶流下来，溪水清澈得能看见石头下面的小鱼在游。溪水不深，但很凉，脚踩进去，整个人都像被整个浸进去了。

我脱掉鞋，卷起裤腿，坐到一块光滑的石头上。水流从我的脚踝上滑过，像无数只小手在轻轻抚摸皮肤。夏天在城里被太阳烤得发烫，现在这水，像要把我从头凉到脚。

我靠在溪边的大石头上，闭上眼睛。头顶是茂密的树冠，阳光被叶子筛得碎碎的，像洒在身上的金粉。耳边是风吹树叶的声音，夹杂着小鸟的鸣叫，还有远处隐约传来的羊羔咩咩声。

这就是秦岭的夏天吗？

我小时候就听过秦岭的故事：它是南北分界线，夏天凉，冬天暖，大熊猫、金丝猴、羚牛……这些传说里的精灵，就藏在这些山里。

我睁开眼，发现身边多了一只小动物。它是条小花蛇，身体细长，颜色像落叶的颜色，安静地躺在石头上看我。它不逃，也不咬，只是用冷冷的眼睛看着我，像在说：“你来了。”

我轻轻伸出手，它没有动，只是尾巴轻轻扫了一下我的手背。那一下凉凉的，像在提醒我：这里不欢迎热浪。

我笑起来。原来被蛇看到，也是种缘分。

从这里往下走，路边突然出现一棵老树。树干粗得要好几个人才能合抱，树皮像被岁月刻满皱纹。树上结满了野果，红彤彤的，亮得晃眼。我走过去，伸手摘了一颗。

果子入口，甜得发腻，却又带着一点青涩的苦。不是我们见过的那种甜水果，这是秦岭特有的“板凳果”。传说古时候有人坐在这棵树下歇脚，屁股被板凳硌得发疼，就叫它“板凳果”。其实酸酸甜甜的，吃多了就上头。

我摘了好几颗，塞在包里。山里的野果，总要带点回去，让家里人尝尝。

吃完果子，我又继续走。溪水越来越宽，变成了两条并行的溪流，中间隔着一块石头滩。石头滩上，野花开得更密了。粉色的、紫色的、红色的，像一场没有结束的梦。

我坐在滩上，把鞋袜都脱掉，光着脚在石头上走。水流过我的脚趾缝，凉得让人想喊出来。夏天终于被这山里的水，软软地按住了。

远处，传来几声悠扬的呼喊。我转头，看见几个背着背包的年轻人。他们在溪边嬉戏，一个女孩子大声喊：“蝴蝶飞！快来玩呀！”

我走过去，他们都认识我。原来我们是老朋友，几个月前在山里聊天时约好的：夏天到了，我来秦岭，她们也来。

现在，大家都来了。

我看着她们，她们也看着我。她们也光着脚在水里跑，笑得像孩子。风把她们的头发吹乱，溪水把她们的裤腿泡得湿透。

我们就这样一直玩到天黑。

天黑的时候，我们找了一处平坦的地方生火。火光映在脸上，暖暖的，却不会烫。火堆旁，有我们采的野果，有我们从山上带来的凉气。我拿出一颗板凳果，递给旁边的小朋友。

“吃啊，这是秦岭的夏天味道。”

她咬了一口，眼睛亮起来：“好甜！比城里的水果好吃多了！”

我点点头。夏天在城里总是热得让人喘不过气，可秦岭的夏天，是另一种味道。

它不是冷，而是**清醒**。它把一切都洗得干干净净，让你看到自己的影子。

我坐在火堆旁边，看着山下的灯火慢慢亮起。秦岭的夏天就这样过去了：不是热得发烫，而是凉得让人心都干净了。

后来，我们一起下山。月光下，溪水还叮叮咚咚地响，像在和我们道别。

我走着走着，忽然想起小时候妈妈常说的一句话：

“山里有水，就有清凉。”

现在我懂了。

清凉，不是温度，而是**心里的温度**。

它不烫手，却能把你从头凉到脚。

秦岭的夏天，就这样轻轻地来了，又轻轻地走了。

可它留下的东西，会一直陪着你。
